Recommended Workflow

  •  Create some zone objects using 'LSS Zone' tool
    •  'Draw Contour' method
    •  'Pick Floor Face' method
    •  Cut openings
  •  Copy the identical zones using 'Move' tool
    •  copy within one floor
    •  copy identical floors
  •  'Rebuild' previously copied zones
  •  Adjust identity attributes of copied zones
    • room number
    • floor number
  •  Attach labels
  •  Generate reports using 'List Zones' command
    • rooms list
    • finishing materials list
    • floors list
    • etc

Note

'View/Edit Properties' dialog rebuilds selected zones in case if non-identity transformation of a zone group detected. That's why 'Rebuild' step is actually optional. But it is also important to be aware of in case if some of zone's internal elements (floor, ceiling, wall) were edited manually using SketchUp tools.
