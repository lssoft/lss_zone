Link Openings

Finds adjacent openings of selected zone objects

Tool Operation

  1. select zone objects
  2. open tool dialog
  3. adjust check distance between openings
  4. launch linking process

Advanced Operations

When links graph is finished it is possible
  • to move its nodes
  • to build links graph in an active model

Post Processing

It is possible to move spheres, which represent graph nodes, inside graph group in a model after its creation, then it is possible to refresh links.
To refresh links after moving spheres inside a group:
  • select graph group in a model
  • right-click on it and choose "Refresh Links Graph" from pop-up menu or select "Plugins>LSS Zone>Link Openings>Refresh Links Graph" from main menu

Note

The additional result of linking openings is that each adjacent opening becomes marked as internal. Each zone stores internal and external openings area separately so linking process affects internal and external openings area obviously. It is also possible to launch 'Recalculate' or 'Rebuild' commands to ensure, that internal openings area and external openings area of selected zones are actual.
