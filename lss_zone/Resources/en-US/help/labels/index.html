Attach Labels

Attaches labels with zones' attributes (number, name, area etc) to selected zones.

Tool Operation

  1. select zone objects
  2. open tool dialog
  3. select existing label template from a drop-down list
  4. set appropriate layer for labels
  5. attach labels

Advanced Operations

It is also possible to edit label template or create a new one.
Label template is just a string (it can be multiline if necessary), which has some mentions of zone attributes prefixed by a '@' sign and may have some explanatory information as well.
Refer to 'Attributes List' chapter of 'Help System' in order to find out what kind of properties can be displayed in a label.

Note

'Rebuild' command refreshes label's contents as well.
