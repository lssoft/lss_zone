View/Edit Properties

Displays 'Properties' dialog.

Command Operation

  1. open dialog
  2. select zone object(s)
  3. view/edit properties by changing values of input boxes
  4. apply settings

Advanced Operations

It is possible to unselect and select other zones while dialog is active. Dialog contents refreshes automatically after selection changes.
If 'Rebuild' option is enabled, then 'Apply' re-generates zone object(s) in a batch manner (it literally launches 'Rebuild' command after changing attributes' values.

Note

Properties dialog has two different display modes: 'basic properties' and 'all properties'.
'Basic Properties' mode displays identification and geometric properties of a zone only. The order of properties is basically the same as in main 'Zone Tool' dialog.
'All Properties' mode displays all available zone properties. It is even possible to attach new custom properties and/or erase existing properties in this mode. Note that 'Rebuild' command restores 'basic properties' of a zone even after their erasing, because 'basic properties' are hard-coded inside zone entity class.
Ability of non-basic attributes erasing gives opportunity to erase previously attached labels if needed.
