Note

It is possible to edit 'area' element manually using native SketchUp tools, then rebuild zone object in order to refresh 'floor', 'walls' and 'ceiling' elements. Note that 'Rebuild' command causes manual modifications loss of 'floor', 'walls' and 'ceiling' elements if any.
If it is necessary to take in account manual modifications of mentioned above elements, then it is necessary to use 'Recalculate' command.
